# 🛍️ Nxt Trendz - E-commerce App

## 📌 Overview

A React-based e-commerce application where users can browse products, apply filters, and view product details.

---

## 🚀 Features

* 🔍 Search products
* 🎯 Filter by category & rating
* 🔄 Sort products (Price High/Low)
* 📄 Product details page
* ➕➖ Quantity control
* 🔁 Similar products
* ⏳ Loader & ⚠️ Error handling

---

## 🛠️ Tech Stack

* React.js
* JavaScript
* CSS
* React Router
* REST API
* Cookies

---

## 🌐 API

* https://apis.ccbp.in/products
* https://apis.ccbp.in/products/:id

---

## ⚙️ Run Locally

```bash
npm install
npm start
```
